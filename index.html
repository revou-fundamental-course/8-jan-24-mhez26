Selamat datang di KALPER

Ini adalah web sederhana untuk menghitung luas dan keliling persegi

LUAS PERSEGI

Rumus luas persegi yaitu:

L = S x S

di mana:

L = luas persegi

S = panjang sisi persegi

KELILING PERSEGI

Rumus keliling persegi yaitu:

K = 4 x S

di mana:

K = keliling persegi

S = panjang sisi persegi
